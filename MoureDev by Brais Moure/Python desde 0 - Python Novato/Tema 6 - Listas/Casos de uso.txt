Listas : estas se utilizan cuando tu quieres una estructura
en la que puedes insertar, eliminar, modificar, o acceder a
elementos por el index, ademas de querer datos repetidos

Tuplas : estas se utilizan cuando quieres que tus datos no
cambien es decir que sean inmutables

Sets : estos se utilizan cuando quieres que tus datos no se
repitan, que no haya redundancias

Diccionarios: estos se utilizan cuando quieres que tus datos
esten relacionados con una palabra clave, podrian utilizarse
tambien para manejar datos del tipo JSON
