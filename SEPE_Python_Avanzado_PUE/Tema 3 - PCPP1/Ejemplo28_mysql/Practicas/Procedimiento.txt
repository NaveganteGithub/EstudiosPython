1º Instalamos MySQL

2º Seguimos los pasos de la aplicacion "MySQL 8.4 Configurator" para activar y configurar el servidor.
    La primera vez te pedira que crees la contraseña del root, 
    mi recomendacion es escribirla en un bloc de notas para 
    despues copiarla y pegarla en el campo

3º Comprobar que podemos conectarnos con el cliente de MySQL "MySQL 8.4 Command Line Client"

4º Realizar la conexion con Python.

    4.1 Tendremos que descargar el driver  de Mysql para Python, concretamente la version 2.1.7

        https://downloads.mysql.com/archives/c-python/
        https://downloads.mysql.com/archives/get/p/29/file/mysql-connector-python-2.1.7.zip

    4.1 Instalar el driver.

        pip install mysqlx-connector-python

    4.2 Crearemos una base de datos para hacer la prueba, asi que iniciaremos sesión

        mysql> create database tienda;
        Query OK, 1 row affected (0.01 sec)

        mysql> use tienda;
        Database changed
        mysql> show tables;
        Empty set (0.02 sec)

        mysql> create table PRODUCTOS (
            -> Id integer primary key,
            -> descripcion text,
            -> precio double
            -> );
        Query OK, 0 rows affected, 1 warning (0.03 sec)

        mysql> insert into PRODUCTOS values (1, "Pantalla", 80.50);
        Query OK, 1 row affected (0.01 sec)

        mysql> insert into PRODUCTOS values (2, "Impresora", 40.15);
        Query OK, 1 row affected (0.01 sec)

        mysql> insert into PRODUCTOS values (3, "Raton", 15.14);
        Query OK, 1 row affected (0.00 sec)

5º Para desactivar MySQL tienes que desactivar el proceso en segundo plano.
