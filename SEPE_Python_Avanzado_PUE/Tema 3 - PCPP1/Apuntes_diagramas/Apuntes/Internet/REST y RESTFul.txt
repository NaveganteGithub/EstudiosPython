REST o Representation State Transfer es un estilo de arquitectura cliente-servidor dedicado a sistemas distribuidos.

Las API RESTful Web Service, son APIs web que son implementadas bajo los principios REST, por ejemplo todas las
peticiones se hacen a traves de una URI con la que podemos acceder al servicio o un formato de mensajes
como JSON o XML.

Las operaciones que se sueles usar son:

- GET que equivale a SELECT de SQL aunque si se invoca multiples veces, retorna el mismo resultado
- POST que equivale a INSERT de SQL aunque este al contrario que insert se utiliza para insertar
  grandes cantidades de datos
- PUT que equivale a UPDATE de SQL
- DELETE que equivale a DELETE de SQL
